# NestAI

AI-Powered Real Estate Discovery | Find Rooms, PGs, Flats & Homes

## Project info

AI-powered property discovery that understands your lifestyle. Find rooms, PGs, flats & homes with intelligent recommendations.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To run this project locally:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
